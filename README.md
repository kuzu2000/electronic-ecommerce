 ## Technology Which I Used

 - Nodejs
 - Mongodb
 - Express

 ***Frontend***
 
 - React Js
 - Redux

***Storing Inage*** 

-firebase

<br />

 ## Step to Start Project
 

 
**For Backend**
 1. `git clone https://github.com/MohitSojitra/e-commerce-store.git`
 2. `cd ecommerce-api`
 3. `npm i`
 4. `npm run start`

**For Frontend**

 1. `cd e-comerce-frontend`
 2. `npm i`
 3. `npm run start`

**For Frontend**

 1. `cd ecommerce-admin`
 2. `npm i`
 3. `npm run dev`
